# Веб-сайт Календаря с расписанием 

## Основная идея
Пользователь регистрируется, загружает своё расписание, а затем ежедневно видит актуальные дела в разделе «Расписание на сегодня»,
дополняя их личными заметками и заданиями в отдельной вкладке.

## Цель проекта
Создать интерактивный веб-сайт, который позволит пользователям организовать своё ежедневное расписание и задачи.
Сайт должен предоставлять возможность регистрации, загрузки личного расписания, отображения расписания на текущий день и добавления собственных заданий.
Вся работа осуществляется в рамках одного сайта, через удобную систему вкладок.


## Технологии
HTML / CSS / JavaScript
Python (модуль анализа и ИИ)
VS Code Live Server для локального запуска сайта
JSON/TXT для БД
